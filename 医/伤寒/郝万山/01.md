# 辨太阳病脉证并治法上
## 原文
1. 1 太阳之为病，脉浮，头项强痛而悪寒。
    > * "轻取即得，举之有余，按之少力，如水漂木" - 浮脉
    > * 体表受邪，正气抗邪于表，气血浮盛于外，脉亦浮于表
    > * 浮脉主表，注意要“举之有余(力)”，若中空，为芤脉；若无力，为虚阳外浮，均不属于浮脉
    > * "脉浮"，可以算是表证的最显著的表现，如果在加上其他的在表的证候，那么其病必有表证
    > * "头项强痛"(头痛 + 后颈拘紧不柔)这一点是 太阳 + 表 证的定位性症状，脉浮只是代表表证；
    > * "悪寒"，因阳气为外邪所伤，必不足也，温煦肌肤功能失司，故而怕冷；肾阳不足造成表阳失助，也会导致恶寒，因此，只有三个条件同时出现，才能基本判断是太阳病
    > * 总结："浮"提示邪在表，"头像强痛"提示邪在太阳，"恶寒"提示外邪伤阳(里阳虚)
2. 2 太阳病，发热，<strong>汗出</strong>，悪风，脉缓者，名为中风
    > * "发热"列在前，说明发热在病程早期即出现；
    > * "发热"的机理在于：风邪伤卫阳，风邪属于阳邪，引发<strong>卫阳出现病理性亢奋</strong>，因此表现出发热，这种形式出现的发热为"卫强"；
    > * "汗出"的机理在于：风邪伤卫阳，卫外失司；同时加上风性主疏泄，津液容易蒸发于体表，即为汗 - 太阳中风的突出特征
    > * "恶寒"者，身居密室，闭其门窗，加衣覆被，仍不觉冷减，此为恶寒
    > * "恶风"者，“当风则恶，无风则缓”，此为恶风，两者注意区分(虽然《伤寒》中常常混用)
    > * "脉缓"，在太阳病的大前提下，实际为脉浮缓。此处的缓，不指脉搏节律快慢，而是指脉搏形态松弛柔软，与太阳伤寒中的紧脉相对，出现脉缓：
        >> 1. 无寒邪
        >> 2. 汗出伤营：汗为津液所化，汗出则阴血中津液不足，以至营气不足，故而血管松弛柔软
    > * 总结：风邪袭表、卫强营弱、营卫失和 - 太阳中风基础病机
3. 3 太阳病，或已发热，或未发热，必<strong>悪寒</strong>，体痛，呕逆，脉阴阳倶紧者，名为伤寒
    > * "悪寒"，寒为阴邪，最易伤阳，在肌表，太阳阳气失，则温煦失司，故而最先表现出恶寒，同时恶寒症状最重
    > * "或已发热，或未发热"，此发热机理为：寒郁阳气，阳气积郁到一定程度之后发热。所以有些人发热早，有些人发热晚
    > * "体痛"(第35条会有更详细的描述)，简单而言就是全身痛，太阳经脉循行的地方都表现出疼痛，其机理在于：寒邪伤表使得肌肤气血涩滞，不通即痛，也使得筋脉拘挛，表现为疼痛。 - 这一点是太阳伤寒的突出症状
    > * "呕逆"，一般此证是胃气上逆的表现，这里并不是寒邪直接伤胃，而是因为体表受邪，正气抗邪于表，于是内里慢慢失调，气机的升降失调，临床则表现为食欲不振到呕逆的由浅到深的一系列证候。
    > * 气机失调，在胃为上逆，在脾则不能升清出现下利，因此，太阳伤寒在消化系统上往往表现出从食欲不振到不大便，从呕逆到下利的一系列由浅入深的症状，这是外感病导致的里气升降失调，而不是邪气入肠胃。
    > * "呕逆"，是太阳伤寒的一个副证
    > * "脉紧"，寒主收引，血管高度紧张，因此脉紧，主寒邪盛
    > * "脉阴阳倶紧"，强调寸尺关三脉俱紧才能算典型的太阳伤寒表实证，而如果缺失其一，比如尺脉沉，那就可能伴有少阴的里阳虚
    > * 第 35 条"无汗而喘"一条，说明太阳伤寒会有无汗的证候
    > * 总结：寒邪闭表、卫闭营郁 - 太阳伤寒基础病机
4. 4 伤寒一日，太阳受之，脉若静者，为不传；颇欲吐者，若躁烦，脉数急者，为传也
    > * "静"，不是脉不跳的意思，这里指没有变化，也就是没传变
    > * "颇"字在古代有两个完全相反的意思，一个是"很，甚"，一个是"少，稍微"，《伤寒》中两种都用到。这里是"很"的意思，另一种见"霍乱病篇"中有"颇能食"的另一种用法。
    > * "颇欲吐"标志着邪气传入少阳，因为少阳病是一个胆热气郁的证候，胆热气郁以至犯胃，所以少阳的临床特征就是心烦喜呕(多呕、善呕)，因此《伤寒》中常以有没有呕吐这一条来提示少阳病的存在不存在，如《伤寒》第61
    > * "若躁烦"，"躁烦"强调了肢体躁动不宁，这是三阴(重)证的临床表现之一。但是这里如果结合下一条来看，郝认为更应该是烦躁，是阳明病，里热、里实，以至上扰心神出现烦躁，大承气汤主之
    > * "脉数急"，与"脉若静"成对比，指脉象变化，意味着邪气传经
5. 5 伤寒二三日，阳明、少阳证不见者，为不传也
    > * 这一条与上一条两条结合起来的重点是：辨传经与否的依据不在于病程天数，而在于临床脉证的变化 
6. 6 太阳病，发热而渴，不恶寒者，为温病。若发汗已，身灼热者，名风温。风温为病，脉阴阳俱浮，自汗出，身重，多眠睡，鼻息必鼾，语言难出。若被下者，小便不利。直视失溲，若被火者，微发黄色，剧则如惊痫，时瘛疭，若火薫之。一逆尚引日，再逆促命期
    > * 此条和太阳中风的发病机理类似，属于温热邪气伤肌表，损阴液，以至体内阳气病理性亢奋  
    > * 温邪常伴有风邪，所以在体表的初期阶段，人可能会有短暂的、轻度的怕风
    > * 如果从证候的表现上，"发热而渴，不恶寒者"与太阳病总提纲冲突，所以此条似乎不应该归类于太阳病，从今天的角度来看，这完全可以归类于温邪上受、首先犯肺的手太阴温病(郝万山)，应该是仲景看到了温邪伤阴液的证候，但是局限于时代而没有归类
    > * 如果从所有证候来看，温病确实兼有两者的证候，发病于表(太阳)，伤肺阴(太阴)
    > * 《伤寒》把风温、温疟、瘟疫这些病名相互并列，郝万山认为风温是一个独立的病名，不一定是温病是误用辛温发汗造成的变证。
    > * 风温的证候是"发汗，身灼热"，即高热，汗出而热不退，临床表现"脉阴阳浮"，注意这里的浮主热而不是主表，里热盛，鼓动气血，气盛血壅，血管扩张，所以这里应该是"脉浮而滑数"
    > * "自汗出"指里热逼迫津液外越；"身重"指热邪壅滞气机；
    > * "多眠睡，鼻息必鼾，语言难出"为热扰心神，热盛神昏；温病在传变的过程中，温邪上受，首先犯肺，逆传心包，所以这一句实际就是后世温病家提出的“逆传心包”(规律)的一个依据
    > * "若被下者，小便不利"，如果使用泻下的方法的话，结果是"小便不利"，其机理是："小便不利"是误下之后，下焦阴伤，化源不足，即小便少，而不是尿道涩痛。
    > * "直视"指两眼呆滞凝滞无神，此下焦肝肾阴伤的表现 - 下焦真阴耗伤，肝肾阴伤，目睛失养，即两眼呆滞凝视无神；"溲"一般指小便，而"后溲"、"大溲"才能称之为大便。此处"失溲"指小便失禁。而小便失禁是热盛神昏，膀胱失约的缘故
    > * 所以综上，"小便不利"指阴液不足，化源枯竭，于是尿少；而"失溲"则表示小便失禁
    > * "若被火者"，指如果误用火疗(意在出汗)，"微发黄色"，轻(症)的就会出现发黄，这个发黄的病机是热伤营血，营气不布(正常人脸色红润正是因为营血正常输布的表现)，火热内伤营血后，正常的营气不能输布导致发黄
    > * "剧则如惊痫，时瘛疭"，"疭"是肢体的伸展，"瘛"是肢体的收引，于是"瘛疭"即肢体一收一伸，在现代的语境中就是抽搐，就像惊(小儿抽搐)痫(成人抽搐)，此为热盛动风
    > * "一逆尚引日，再逆促命期"，一次错误治疗尚且能多活几天，多次错误就是缩短病人的生命了
7. 7 病有发热恶寒者，发于阳也；无发热恶寒者，发于阴也。发于阳，七日愈；发于阴，六日愈。以阳数七，阴数六故也
    > * 这一句"发于阴阳"，不同注家有不同的解释，这里只简单记录理解，其他的具体见视频第九讲：
    >> 1. 阳者，三阳之证，阴者，三阴之证；阳证以发热为主，阴证以恶寒而不发热为主要特征。这样理解的话与"发于阴，六日愈"似乎有冲突，且太阳病发热恶寒通见，也不太好解释。这种解释部分符合临床实际
    >> 2. 阳者，发于太阳，阴者，发于少阴。这种解释也部分符合临床，但是少阴病中的少阴寒化证(真阳衰)，六天同样无法自愈，与"六日愈"也不符合
    >> 3. 阳者，太阳中风也，阴者，太阳伤寒也。这一解释可以很好的与下文契合，因此这是最符合原文的一种解释
    > * 阳七阴六的节律，具体也可以见视频第九讲
8. 8 太阳病，头痛至七日以上自愈者，以行其经尽故也。若欲再经者，针足阳明，使经不传则愈
    > * "七日自愈"的意思是，以头痛为代表的太阳病，在没有经过治疗，也没有合并证和并发证的情况下，第七天即可自愈，"以行其经尽故也"，即此病的自然病程结束。医生在此的意义是截断病程，减少痛苦
    > * "若欲再经"这里一般理解为进入第二个病程，即第一轮的自然病程没好，会进入第二轮的自然病程，当然这里病即可以停留在太阳经，也可以传变入其他经。但是，不管有没有传变，仲景认为可以通过针足阳明(足三里穴)以截断病程
9. 9 太阳病，欲解时，从巳至未上
    > * "欲解时"指一天中病最容易好的时间，也可以说是一关(要么好要么"再经")，六经各有自己的"欲解时"
    >> 1. 少阳病在凌晨三点年至上午九点(寅时-辰时)，这段时间自然界阳气生发，少阳阳气在这段时间最为旺盛，少阳病病症的加重也在这个时候。少阳阳气旺，正邪斗争激烈，口苦、咽干、头晕加重，如果本来少阳气虚，那么情绪低落，精神忧郁；当然这个时候也是少阳病最容易好的时候，内阳借外阳之势，驱邪于外。故而少阳病"欲解时"在清晨，同时少阳病正邪斗争最激烈的时候也在清晨
    >> 2. 太阳病在中午(巳时-未时)，因此在这段时间也容易出现汗出、热退的现象
    >> 3. 阳明病在下午(申时-戌时)，因此如果今天没有到节律，那么下午往往出现高热的表现，即"日晡所发潮热"，其他症状有：谵语，腹满痛，绕脐痛等。阳明病在这个时候用药效率最好
    >> 4. 太阴病在夜间(亥时-丑时)
    >> 5. 少阴病在夜间(子时-寅时)
    >> 6. 厥阴病在夜间(丑时-卯时)
10. 10 风家，表解而不了了之者，十二日愈
    > * 七日节律是一个一般性的规律，此条讲的是特殊性的规律
    > * 风家，容易太阳中风的人，此类人平素表阳多虚，所以一有风吹草动就容易中风。
    > * "不了了"，算是仲景时代的一种地域性方言。"了"字意思即痊愈，同样，后文中如"若不差"中的"差"字，也是痊愈的意思。这句"不了了"大意就是没有完全好，身上还有点不舒服的地方。表邪已解，烧也退了，头项强痛也缓解了，为什么还有不舒服呢？因为虚人外感病邪虽退，但是正气尚未恢复，故多出5天的恢复时间(五脏系统，5天一候)
11. 11 病人身大热，反欲得衣者，热在皮肤，寒在骨髄也；身大寒，反不欲近衣者，寒在皮肤，热在骨髄也
12. 12 太阳中风，阳浮而阴弱，阳浮者，热自发，阴弱者，汗自出，啬啬悪寒，淅淅悪风，翕翕发热，鼻鸣干呕者，桂枝汤主之
    > * 此条时太阳中风的另一种说法
    > * "阳浮而阴弱"，可以指脉象，轻取见浮，重按松柔；也可以指病机
    > * "阳浮者，热自发，阴弱者，汗自出"，注意这里阴阳因果的前后关系。因阳浮于表，所以说热自发；而因为汗自出，所以才阴弱
    > * "啬啬悪寒，淅淅悪风，翕翕发热"，恶风寒，太阳中风证的基本病机
    > * "鼻鸣干呕"是太阳中风的兼证，鼻鸣是风邪袭表，风邪上壅，肺窍不利的表现；干呕的机理同太阳伤寒的"呕逆"
    > * 辨出病证后，就是治法。既然是风邪袭表，卫强营弱，营卫失和，那么在治疗上，就是<strong>解肌去风，调和营卫</strong> - 桂枝汤
13. 13 太阳病，头痛，发热，汗出，悪风，桂枝汤主之
    > * 这条的意思呢，凡是太阳病，见到头痛，发热，汗出，悪风等证候，都可以用桂枝汤。这其实扩大了桂枝汤等使用范围，这样桂枝汤就不仅仅可以治疗太阳中风(第 2 条，第 12 条)，或者说，太阳中风是桂枝汤治疗的范围之一
    > * 抓主症，对症定药，而不是拘泥于教条。当然这需要在丰富经验的前提下
14. 14 太阳病，项背强几几，反汗出悪风者，桂枝加葛根汤主之
    > * "几几"，郝万山认为这个字是通假字，通"掔掔"(古音同"紧"，今音同"迁")，意思和今天的"紧紧"差不多的意思。因此"项背强几几"就是项背肌肉拘谨不柔和。这句提示太阳经上有邪气
    > * "反汗出悪风"，有汗、恶风，表示风邪在经，经气不利。如果只用桂枝汤的话，那么解不了经上的邪气，所以需要加一味葛根
    > * 桂枝汤加葛根的主体思路是解肌祛风，疏通经脉
    > * 一般经脉拘挛，多半同时伴随着津液不能滋润的因素

15. 15 太阳病，下之后，其気上冲者，可与桂枝汤，方用前法；若不上冲者，不得与之
    > * "其気上冲"，郝万山这个气指的是太阳阳气，上冲说明太阳正气还能至体表去抵抗风邪，临床表现为表证仍然存在。如果这样解释就能与后文"若不上冲者，不得与之"一起说得通了：太阳正气不上冲，即邪不在表，提示已经传变，故而不能再用桂枝汤

16. 16 太阳病三日，已发汗，若吐、若下、若温针，仍不解者，此为壊病，桂枝不中与之也，観其脉证，知犯何逆，随证治之。桂枝本为解肌，若其人脉浮紧，发热汗不出者，不可与之也。常须识此，勿令误也
    > * 此为桂枝汤禁忌证之一
    > * "解肌"与麻黄汤的发汗散寒相对，桂枝汤解肌祛风
    > * 太阳伤寒，无汗、表闭，故以表实名之；太阳中风，有汗、营阴不足，故以表虚名之。这里两个概念相对，并不是真正的气虚/实(精气夺则虚)
    > * 如果太阳伤寒证误用桂枝汤，因汤中芍药酸敛阴柔，容易使得阳气进一步内郁积化热

17. 17 若酒客病，不可与桂枝汤，得之则呕，以酒客不喜甘故也
    > * 此为桂枝汤禁忌证之一
    > * "酒客"，常嗜酒之人，这种人体质湿热内盛(舌红、苔黄厚而腻)。湿热内盛阻遏营卫气血，再某些临床症状上类似太阳中风。所以这种情况需要留意舌苔是否红腻，同时，脉也不是表证的象，而是脉滑而数，因此这种情况不能错误地用桂枝汤去治，应该清热里湿，调和中州
    > * "不喜甘"，甘能增湿，桂枝汤性偏温，温能助热，温甘必然加重中焦湿热，胃气上逆，所以得之则呕
18. 18 喘家，作桂枝汤加厚朴、杏子佳
    > * 桂枝汤加厚朴杏子汤的一个适应证
    > * "喘家，作桂枝汤加厚朴、杏子佳" 这是郝万山的断句建议，按照这样断句，就是医生医生给病人开方的意思，而如果是"喘家作"，容易理解成哮喘发作的意思。《伤寒》中的"作"大多都是开方的意思
    > * 这条指的是新感(太阳中风)引发宿喘(长期哮喘)的病人，风邪容易引起肺气宣发肃降的失调。宜先治疗新感，而加入的厚朴、杏子，则宽胸、利气、降费、平喘，属于辅助调理哮喘之功

19. 19 凡服桂枝汤吐者，其后必吐脓血也
    > * 此为桂枝汤禁忌证之一，起因是毒热内盛
    > * "吐脓血"，说明原先体内有化脓性的病灶，即内痈。体内毒热内盛，腐破血络，使得体内形成化脓性的病灶。毒热内盛，就会发热，逼迫津液外越，即出汗。毒热内盛，阻遏营卫气血，表现出类太阳中风。这时候仍然可以观察到舌红口渴、大便秘结，小便短赤的里热证。此时应当以清热解毒、化脓破腐的方法来治疗
20. 20 太阳病，发汗，遂漏不止，其人悪风，小便难，四肢微急，难以屈伸者，桂枝加附子汤主之
    > * "漏"者，淋漓不止，"发汗，遂漏不止"就是汗出淋漓而不止的意思，为何会漏？可以是误治，可以是药量过之
    > * "小便难"者，汗出不止，阴液化汗而出，不足，故而不见小便
    > * "四肢微急，难以屈伸"，此阴阳两伤：一者阴液被伤，经脉失濡；二者阳气被伤，温煦失司，经脉失温度，如冬天裸露的手指较难屈伸
    > * 此方的基本病机是太阳病，阴阳两伤，表未解。细考此条，只用附子以补阳，何以？因"固阳以摄阴"，古语云："有形之血不能速生，无形之气所当急固"，即大失血后，气随血脱，应用大剂量独参汤益气固阳。不过现代输血技术发达，所以也不需要太拘泥于此。但是此条中的固阳以摄阴的思路仍是我们需要学习借鉴的
21. 21 太阳病，下之后，脉促胸满[men,四声]者，桂枝去芍药汤主之
    > * "满"字在汉时有两种读音：man，三声者，水满；men，四声者，气满，通懑或闷
    > * 何以"满"?胸为气海，当气机不利的时候，即为胸满。故在胸在胁时，一般都是四声的读法
    > * 太阳病，本不应泻下，误下之后，造成了胸闷，是表邪内陷于胸中，使得胸中气机不畅
    > * 何以脉促，表邪内陷胸中，需要鼓应正气以抗邪，所以这个脉促，其实是通过加快心率的方式来抗邪，所以这里的脉象其实应该是脉促但无力
    > * 何以"去芍药"，芍药酸敛阴柔，桂枝汤中起养营敛汗多作用，现在邪气陷于胸中，体表没有"自汗出"的症状，所以不需要芍药。这也是《伤寒》的一个规律：凡胸中气机不畅的，都不怎么用酸敛阴柔的药(白芍)
22. 22 若微寒者，桂枝去芍药加附子汤主之
    > * "微寒"者，脉微、恶寒，这里是以方子中的"附子"来反推是脉微、恶寒，而不是微恶寒
    > * 附子的用法，一定存在肾阳虚，表阳不固的病机。肾阳虚，所以脉微。里阳虚(肾阳虚)，表阳不固，温煦失司，即恶寒


24. 24 太阳病，初服桂枝汤，反烦不解者，先刺风池、风府，却与桂枝汤则愈
    > * 此法针、药并用
    > * "初服"指第一剂第一服
    > * "烦"，代表有热，这里是指的是"烦热"的意思，或者更干脆点即太阳中风的"发热"
    > * 何以"烦不解"？病重药轻，不足以祛除邪气，这种情况名"激惹"
    > * "却"即"再"

42. 42 太阳病，外证未解，脉浮弱者，当以汗解，宜桂枝汤
    > * 表证兼轻度里虚，而麻黄汤发汗力强，易伤正气，此时里又略虚，不宜用麻黄
    > * 此条亦笼统，并未明确说明是中风还是伤寒
    > * "外证"即表证，说明表证未解
    > * "脉浮弱"，"浮"同样表示为表证，"弱"则说明正气轻度不足
43. 43 太阳病，下之微喘者，表未解故也，桂枝加厚朴杏子汤主之
    > * 桂枝汤加厚朴杏子汤的一个适应证
    > * 这条指的是新感(太阳中风)引发新喘(与宿喘相对)的病人，太阳病，泻下本就是一种错误的治法，泻下之后，邪气陷于胸，肺气不利，出现轻喘，所以用此方可同事治中风与新喘
44. 44 太阳病，外证未解，不可下也，下之为逆。欲解外者，宜桂枝汤
    > * "不可下"提示里有实热，"下"指泻下
    > * 此条到意思，外有表证，内忧实热，应当先解表后攻里。为何？攻里的药物，引体内之正气趋内，因此体表的正气必然会被抽调，那么就容易使得表邪内陷，使得病情趋于复杂。所以这种时候应当先解表。
    > * 这里同样不用管伤寒或者中风，都应用桂枝汤。同样是因为麻黄汤纯辛温，发汗力强，伤阴液，这往往会使得里热更甚 - 伤津化燥助里热
45. 45 太阳病，先发汗不解，而复下之，脉浮者不愈，浮为在外，而反下之，故令不愈。今脉浮，故在外，当须解外则愈，宜桂枝汤
    > * "而复下之"，这一句中的"复"和后面的"而反下之"中的"反"字同意，都是反复的意思

53. 53 病常自汗出者，此为栄気和，栄気和者，外不谐，以卫気不共栄気谐和故尔。以栄行脉中，卫行脉外，复发其汗，栄卫和则愈，宜桂枝汤
    > * 这一条，没有太阳中风的典型临床表现，只是一个"常自汗出"，也没有脉浮、恶风寒等，所以说明这个病不是外邪所致
    > * "自汗出"的机理是：卫阳不足，卫外功能失司，营卫失调，卫阳不能固护营阴，以至于营阴外越，即自汗出
54. 54 病人蔵无他病，时发热、自汗出而不愈者，此卫気不和也。先其时发汗则愈，宜桂枝汤
    > * 这条同样只有一个临床表现："时发热、自汗出而不愈"，也没其他什么毛病，所以也不是外感病
    > * 治法是在规律性发汗之前(或者间歇期间)服桂枝汤

56. 56 伤寒不大便六七日，头痛有热者，与承気汤。其小便清者，知不在里，仍在表也，当须发汗。若头痛者，必衄，宜桂枝汤
    > * "宜桂枝汤"，郝认为应该接在"当须发汗"之后
    > * "不大便"也可以是阳明燥热内盛，所以要观察小便。如果是阳明病，那么小便必短赤，而这里"小便清"提示并非在阳明。同样"不大便"也说明里气虚弱，不宜用麻黄汤
57. 57 伤寒发汗已解，半日许复烦，脉浮数者，可更发汗，宜桂枝汤
    > * "数"，主发热

62. 62 发汗后，身疼痛，脉沈遅(沉迟)者，桂枝加芍薬生姜各一両，人参三両，新加汤主之
    > * 在《伤寒》中，脉迟主营气不足，血少。因此这里的"身疼痛"，其实是营血不足，肌肤失养，不荣则痛，即"虚则痛"、"失养则痛"
    > * 这是与"不痛则痛"相反的机理，现实也存在这样的情况，所以需要辩证

63. 95 太阳病，发热、汗出者，此为栄弱卫强，故使汗出。欲救邪风者，宜桂枝汤
    > * 这条也是太阳中风的适应症之一







## 概说
* 成因：
    * 常见：风寒外袭
    * 少见：少阴寒盛伤阳，正气抗邪，阳气有恢复(爆发)的倾向，于是祛邪达表，脏邪还腑，阴病出阳
* 病位：(不涉小肠经腑)
    * 足太阳膀胱经
    * 足太阳膀胱腑
    * 肺主皮毛，外邪在表，也会引起肺的宣发肃降失调
* 生理：
    * 经：
        * 循经从头到表，行于头、项、背
        * 其脉上连风府，和督脉相通，下络腰肾，与肾相连
        * 经别散布于心，故有时可以出现热扰心神，以至发狂等的精神症状
    * 脏腑：主司气化
        * 化生阳气：肾阳温煦之下，产生阳气，名太阳之气，输布于体表
            > "三焦膀胱者，腠理毫毛其应"，因太阳之气经足太阳膀胱经和三焦输布于体表(腠理毫毛)之故也
        * 参与水液代谢：
            * 排除废水 - 功能失，则对应小便不利
            * 化生津液，并输布上承 - 功能失，则对应口渴、消渴、烦渴
    * 阳气 - “太阳主表而统营卫”，因此在体表的阳气其实就是卫气
        * 生成与布达：化生于下焦，补充于中焦，宣发于上焦
            > * 补充于中焦：借助脾胃产生的水谷精微不断补充
            > * 宣发于上焦：借助肺的宣发作用才能均布于体表
        * 功能：- "肥腠理，司开阖，卫外而为固"
            * 温养肌表
            * 管理汗孔开合，调节体温(阴阳于五行的结合)
            * 防御外邪

## 太阳病证候分类
### 本证
* 发生在本经、本脏腑的病变
* 又分表证、腑证：
    * 表证(经证，非专指经脉)：邪气在浅表
        * 有汗(汗孔开而不合) - 中风 - 桂枝汤
        * 无汗(汗孔闭而不开) - 伤寒 - 麻黄汤
        * 麻桂合方，小汗方 - 桂麻各半汤，桂二麻一汤，桂枝二越婢一汤
    * 腑证：表邪不解，邪气循经入里
        * 气分证：太阳蓄水 - 五苓散
            * 气化不利，小便不利；
            * 津液化生不利，不能输布上承，于是渴、消渴、烦渴；
            * 水蓄下焦，阻遏下焦气机，于是少腹苦里急
        * 血分证：太阳蓄血 - 热势急，淤血轻，主泄热辅化淤，核桃承气汤；热势缓，淤血清，破血逐瘀，抵挡丸；若俱缓，抵挡丸
            * 表邪循经入腑化热，热与血结于下焦，于是少腹急结或者硬满
            * 同时太阳经别散步于心，热可能循经入心，以至于热扰心神，出现狂等精神症状

### 变证
* 太阳病误治而来或者失治之后，使得临床证候发生异常变化，而新的辩护不属于六经病的，不能用六经名来命名
* 此证变化多端，或热或寒，或虚或实，或表或里，“观其脉证，知犯何逆，随证治之”

### 类证
* 属于杂病，在此种病程中，有些症状类似于太阳病，故而出现于太阳病篇，以示鉴别

### 太阳病的分类
| 种类 | 症状|
| :----: | :----: |
| 太阳伤寒 | <strong>恶寒</strong>、发热、无汗、体痛、脉浮紧 |
| 太阳中风 | <strong>汗出</strong>、发热、恶风，脉浮缓 |
| 太阳温病 | 发热而渴但是不恶寒 |
| 风温 | 高热、汗出而热不退 |
* 以发热为特征的
* 前二为外(风)寒引起，后二是温(热)邪引起的

## 方
### 桂枝汤
1. 桂枝三两，芍药三两，甘草二两，炙，生姜三两，切，大枣十二枚，擘。此五味，㕮咀三味，以水七升，微火煮取三升，去滓，适寒温，服一升。服已须臾，啜热稀粥一升余，以助药力。温覆令一时许，遍身，微似有汗者益佳，不可令如水流漓，病必不除。若一服汗出病差，停后服，不必尽剂。若不汗，更服依前法。又不汗，后服小促其间，半日许令三服尽。若病重者，一日一夜服，周时观之。服一剂尽，病证犹在者，更作服，若不汗出，乃服至二三剂。禁生冷、黏滑、肉面、五辛、酒酪、臭恶等物。
    > * 两组配伍，一组辛(桂枝、生姜)甘(甘草、大枣)化阳，桂枝、生姜温辛解表，祛风散寒，扶助卫阳；一组酸(芍药)甘(甘草、大枣)化阴，助养营阴，收敛汗液 - 双向调节作用
    > * 此 5 味，有 4 味与食物相关(除芍药)，能开胃气，和胃气。脾胃是气血生化之源，营卫化生之源。因此，此方适量加减，即可调和中州，调和脾胃进一步达到调和阴阳、气血、营卫
        >> * 芍药加倍再加饴糖即小建中汤，温中补虚，和里缓急
        >> * 桂枝加龙骨牡蛎汤，主治女子梦交，男子失精
    > * 此药养正力大，发汗力弱，所以在用后护理上，就有特殊的要求
    > * "㕮咀"，本意咀嚼，在这里是把药捣碎，原因是古代不像现在这样有切片机器，需要把桂、芍、甘三味药捣碎
    > * "以水 7 升"，在汉代 1 升相当于现在的 200ml；"微火煮取三升"，小火煮得剩 3 升，去渣，放到不凉不热的时候，一次只吃一升(上述药量，是三次的治疗量)
    > * "服已须臾，啜热稀粥一升余，以助药力"，用药后片刻需要喝热粥以助药力，一来自发汗会损阴液，故而以粥补之；而来以热粥之热继续发汗，鼓舞胃阳进而振奋卫阳，于中焦补充太阳阳气
    > * "温覆令一时许"，盖上被子保温一个时辰(发汗)，"遍身，微似有汗者益佳，不可令如水流"，需要周身发汗，包括手脚，此发汗的第一个要求；第二个是出小汗、微汗，而不能大汗淋漓；第三个即"一时许"，最好持续出一段时间的汗
    > * "不必尽剂"一服吃完就好了的话不必把剩下的都喝掉；若一服后未出汗，"更服依前法"(第二服)；"又不汗"，第二服后仍不出汗，就需要缩短服药时间了，半日内三服尽用。"若病重者，一日一夜服，周时观之"，病重的要连续给药，一天一夜时时观察；"服一剂尽，病证犹在者，更作服，若不汗出，乃服至二三剂"一剂不行，两剂、三剂连续跟上
    "> * "生冷"易伤胃阳；"黏滑"不易消化；"肉面"的话需要结合各地平时的饮食习惯，这里也多有不易消化的意思，因人而异；"五辛"就是刺激味的蔬菜，如葱蒜香菜等；"酒酪"指酒类和奶制品；"臭恶"指气味不良等食物。总之就是妨碍胃气的食物都需要注意(因人而异)
2. 桂枝汤的适应证：
   1. 太阳病中风证 - 第2、12、95条
   2. 太阳病，已治未治，见头痛，发热，汗出，悪风 - 第13条
   3. 表证兼里实，先解表 - 第44条
   4. 表证汗下(后)，正气受挫，表邪未解 - 第45、57条
   5. 表证兼轻度里虚 - 第42条
   6. 非外邪所致的营卫失和 - 第53、54条
3. 桂枝汤的禁忌证：
   1. 明确表现出来的太阳伤寒 - 第16条
   2. 内里有热("桂枝下咽，阳盛则毙"，临床表现是舌红、口渴、咽痛、咽喉红肿的) - 第17、19条

### 桂枝汤加葛根：
1. 葛根四两，芍药二两，生姜三两，炙甘草二两，大枣十二枚，擘，桂枝二两，去皮。上七味，以水一斗，先煮葛根，减二升，去白沫，内诸药，煮取三升，去滓，温服一升，覆取微似汗，不须啜粥，余如桂枝法将息及禁忌(注，原文中有麻黄，若果加上麻黄，就是葛根汤而不是桂枝汤加葛根了，所以麻黄应该是传抄错误)
    > * "先煮葛根"因为葛根辛温，而且以古时的用法，多是新鲜即采，没有现在的制片技术。所以先煮，减少其性温燥烈之性，同样因为葛根辛温，所以不需要另服热粥
    > * 此方的思路主要思路是解肌祛风，疏通经脉，此药思路是芍药、甘草的量稍微重一点，用酸甘化阴来滋养经脉。郝万山的用量是：葛根20-30(40)，芍药30，桂枝10，甘草6-10
    > * 此方同样可以治疗颈椎病的头项肌肉酸疼，不过用量需要加减
2. 解肌祛风，疏通经脉 - 第14条：
3. 葛根作用有三：一，升阳发表，以助解表；二，疏通经脉；三，升津液，起阴气，滋润经脉

### 桂枝汤加厚朴杏子：
1. 适应证
    * 第18条 - 新感引动宿喘
    * 第43条 - 新感引动新喘

### 桂枝加附子汤：
1. 桂枝三两，芍药三两，炙甘草三两，生姜三两，大枣十二枚，擘，炮附子一枚，去皮，破八片。上六味，以水七升，煮取三升，去滓，温服一升，将息如前法
2. 适应症 - 第20条
3. 此方精髓：固阳以摄阴

### 桂枝去芍药汤、桂枝去芍药加附子汤：
1. 桂枝去芍药汤：
   * 桂枝三两，炙甘草二两，生姜三两，大枣十二枚，擘。上四味，以水七升，煮取三升，去滓，温服一升，将息如前法
   * 此方祛除芍药，方性纯辛温，故而对于心阳虚的病人效果显著
   * 适应症 - 第21条
2. 桂枝去芍药加附子汤：
   * 桂枝三两，炙甘草二两，生姜三两，大枣十二枚，擘，炮附子一枚，去皮，破八片。上五味，以水七升，煮取三升，去滓，温服一升，将息如前法
   * 此方另加附子，所以在补心阳的基础上以附子补肾阳，故而对遇寒则发，夜间发作的冠心病有显著的疗效
   * 适应症 - 第21条

### 桂枝加芍薬生姜各一両，人参三両新加汤：
1. 桂枝三两，芍药四两，炙甘草二两，人参三两，大枣十二枚，擘，生姜四两。上六味，以水一斗二升，煮取三升，去滓，温服一升
2. 适应症 - 第62条
3. 此方可以治疗营血不足，肌肤失养的证候，属于虚则痛，失养则痛大类
4. 加重芍药以养血柔筋，缓痉解痛；加重生姜引药达表；加人参是益气养血
5. 此方对营血不足，肌肤失养的有很好的疗效；而对于营血不足，内脏失养的，用八珍汤或者人参养荣汤
