# 太阳病伤寒证
## 原文
1. 3 太阳病，或已发热，或未发热，必<strong>悪寒</strong>，体痛，呕逆，脉阴阳倶紧者，名为伤寒
    > * "悪寒"，寒为阴邪，最易伤阳，在肌表，太阳阳气失，则温煦失司，故而最先表现出恶寒，同时恶寒症状最重
    > * "或已发热，或未发热"，此发热机理为：寒郁阳气，阳气积郁到一定程度之后发热。所以有些人发热早，有些人发热晚
    > * "体痛"(第35条会有更详细的描述)，简单而言就是全身痛，太阳经脉循行的地方都表现出疼痛，其机理在于：寒邪伤表使得肌肤气血涩滞，不通即痛，也使得筋脉拘挛，表现为疼痛。 - 这一点是太阳伤寒的突出症状
    > * "呕逆"，一般此证是胃气上逆的表现，这里并不是寒邪直接伤胃，而是因为体表受邪，正气抗邪于表，于是内里慢慢失调，气机的升降失调，临床则表现为食欲不振到呕逆的由浅到深的一系列证候。
    > * 气机失调，在胃为上逆，在脾则不能升清出现下利，因此，太阳伤寒在消化系统上往往表现出从食欲不振到不大便，从呕逆到下利的一系列由浅入深的症状，这是外感病导致的里气升降失调，而不是邪气入肠胃。
    > * "呕逆"，是太阳伤寒的一个副证
    > * "脉紧"，寒主收引，血管高度紧张，因此脉紧，主寒邪盛
    > * "脉阴阳倶紧"，强调寸尺关三脉俱紧才能算典型的太阳伤寒表实证，而如果缺失其一，比如尺脉沉，那就可能伴有少阴的里阳虚
    > * 第 35 条"无汗而喘"一条，说明太阳伤寒会有无汗的证候
    > * 总结：寒邪闭表、卫闭营郁 - 太阳伤寒基础病机
2. 31 太阳病，项背强几几，无汗，悪风，葛根汤主之
    > * 伤寒兼证之一
    > * 与第14条不同，此处是无汗；此外，此条的"悪风"恐怕应是"恶寒"
    > * 桂枝汤加葛根治疗风邪在经，太阳经气不利；葛根汤则主治寒邪在经
3. 32 太阳与阳明合病，必自下利，葛根汤主之
    > * "自下利"，因正气抗邪于表，不能固护于里，里气升降失常，非阳明里实
4. 33 太阳与阳明合病，不下利而呕者，葛根加半夏汤主之
    > * 同样是正气抗邪于表，不能固护于里，里气升降失常(上逆为呕，下降为利)
    > * 加半夏，兼以和胃降逆止呕
5. 35 太阳病，头痛发热，身疼腰痛，骨节疼痛，恶风无汗而喘者，麻黄汤主之
    > * "头痛发热，身疼腰痛，骨节疼痛，恶风，无汗而喘" - 伤寒八证，实际可浓缩为三组：
    >   1. "发热，恶寒" - 此为表证，寒邪伤阳，留于肌表，故而恶寒先见，寒邪闭表，阳郁于内，故而发热随后
    >   2. 头身腰骨(关节)诸痛 - 寒主收引，又主痛，寒伤肌表，使得气血涩滞，筋脉拘挛，不通则痛，故而全身疼痛，<strong>这是太阳伤寒最突出的症状</strong>
    >   3. "无汗而喘" - 寒邪闭表，故无汗，肺主汗孔开合，腠理闭塞，必然引起肺的宣发肃降失调，因此肺气上逆作喘
6. 36 太阳与阳明合病，喘而胸満者，不可下，宜麻黄汤
    > * "阳明"，这个阳明并非指阳明里实证，如果是里实(热)，那么张仲景应该会用桂枝汤先解表(见条文第44)，此处应该是阳明经脉为寒邪所伤，阳明经脉所伤表现的证候，见下文的补充部分
    > * "喘而胸満"，肺气不利，如果出现这种证候，一般就表示太阳表邪偏重了，所以不用葛根汤而用麻黄汤，且麻黄汤兼有宣肺平喘的作用

7. 37 太阳病，十日以去，脉浮细而嗜卧者，外已解也，设胸満胁痛者，与小柴胡汤；脉但浮者，与麻黄汤
    > * 此伤寒表实特殊体质的情况
    > * "脉浮细"，伤寒的脉是紧的，现在由紧变细(由大变小)，说明邪气退
        >>  "(脉)大则病进，小则平" - 《黄帝内经》，这里需要注意与平时的生活工作关联，如脑力劳动者的脉应该较体力劳动者的小
    > * "设胸満胁痛者"，邪入少阳，少阳经气不利，宜用小柴胡汤
    > * "脉但浮者"，十日后，脉象没变的，仍然用麻黄汤
    > * 以上是三种可能出现的情况
8. 38 太阳中风，脉浮紧，发热，悪寒，身疼痛，不汗出而烦躁者，大青竜汤主之。若脉微弱，汗出悪风者，不可服之，服之则厥逆，筋惕肉瞤，此为逆也
    > * 此处"太阳中风"，恐怕是"中风"&"伤寒"的互词，应该更关注实际的临床表现
    > * "烦躁"者，寒邪闭表，阳郁化热，热扰心神。"不汗出"导致了"烦躁"
    > * "脉微弱"，脉薄，是真阳虚衰；"汗出"，阳不摄阴；"悪风"，表阳不足，温煦失司；这些都表示弱阳不胜阴，于是出现肢体躁动不安，为阳虚而躁(少阴证)，所以一定要与大青龙汤本证的热扰心神而出现的"烦躁"区别开来
    > * "厥逆"，手足厥冷，真阳进一步虚衰导致；"筋惕肉瞤"，筋肉俱动(不自主跳动)，筋肉既失阴液濡润又失真阳的温煦
9. 39 伤寒脉浮缓，身不疼，但重，乍有軽时，无少阴证者，大青竜汤发之
    > * "浮"，邪在表；"缓"，结合下文的"身不疼，但重"，说明是主湿的缓，不是中风的缓。此结合起来说明是湿邪在表
    > * "乍有轻时"，为何身体时重时轻？因湿邪欲向里，故时轻。此条说明湿郁肌表
    > * "无少阴证"，这句，说的是没有少阴证的弱阳不胜阴的无意识烦躁("燥烦")，可以反推应当是有邪(湿)闭其表以至热扰心神的这种烦躁，所以同样可以用大青龙汤
10. 40 伤寒表不解，心下有水气，干呕发热而咳，或渴，或利，或噎，或小便不利，少腹满，或喘者，小青龙汤主之
    > * "心下有水气"，此为病机，指心下胃脘部平素就有水饮内留：外有表寒，内忧水饮
    > * "干呕发热而咳"这是小青龙汤的三个主证：干呕是水邪犯胃，胃气上逆；发热是外有表邪；咳，是最主要的主证，因外寒引动内饮，内水外寒合邪，水寒射肺导致肺气失常而咳
    > * 水性润下，变动不居，故而体内的水邪，往往会随着三焦气机四处游走为患，所以见渴、利、噎、少腹满等
11. 41 伤寒，心下有水気，咳而微喘，发热不渇；服汤已，渇者，此寒去欲解也；小青竜汤主之。
    > * "小青竜汤主之"应该接在"发热不渴"后面
    > * "发热"，病邪在表；"不渇"，其病机是寒饮内盛，阳气不运化，不能消水，故而不渴(注意与上一条的"或渴"区别)
    > * 服药后的"渇者"，寒饮温化，津液一时不足(阳热药伤津)，故而口渴
12. 46 太阳病，脉浮紧，无汗，发热，身疼痛，八九日不解，表证仍在，此当发其汗。服薬已微除，其人发烦目瞑，剧者必衄，衄乃解。所以然者，阳気重故也。麻黄汤主之
13. 47 太阳病，脉浮紧，发热，身无汗，自衄者愈
14. 49 脉浮数者，法当汗出而愈。若下之，身重心悸者，不可发汗，当自汗出乃解。所以然者，尺中脉微，此里虚。须表里実，津液自和，便自汗出愈
    > * 禁忌症之一
    > * "法"应该是"理"
    > * "浮"，邪在表，"数"，内有热，此表证之象，理当发汗
    > * "若下之"，如果用了下法，出现身重、心悸、脉微、尺中脉微的情况，不应该再用汗法。《伤寒》中"身重"有两种情况，一种是前文中第6条所示的邪气盛，另一种是这条所示的正气虚。正气虚，能量不足，活动不灵活，即身重；其后"尺中脉微"，鼓动无力，阳虚，心脏失养，故而心悸。总之，这里的两个表现提示了阳虚的情况
    > * 里虚兼表实证，不宜汗法，宜适当调理机体，使得表里皆实，自汗出即愈
15. 50 脉浮紧者，法当身疼痛，宜以汗解之。仮令尺中遅者，不可发汗，何以知然？以栄気不足，血少故也
    > * 禁忌症之一
    > * "法"应该是"理"
    > * "假令尺中迟者"，尺脉迟，非浮紧，不可发汗，因营血不足，脉行迟滞
16. 51 脉浮者，病在表，可发汗，宜麻黄汤
17. 52 脉浮而数者，可发汗，宜麻黄汤
    > * 51、52两条强调一定要注意脉浮才难用麻黄汤
    > * "数"指发热带来的数，而不是里热的"数"(两者有何不同?)
18. 55 伤寒脉浮紧，不发汗，因致衄者，麻黄汤主之
    > * 此衄者，点滴不畅，方可用麻黄汤，为正邪相斗，正略弱于邪，故以麻黄汤助正气以解表邪
19. 83 咽喉干燥者，不可发汗
    > * 禁忌症之一
20. 84 淋家，不可发汗，发汗必便血
21. 85 疮家，虽身疼痛，不可发汗，汗出则痓(音志)
    > * 禁忌症之一
    > * 此处"痓"实应为痉，即以抽搐为主要的临床表现(角弓反张，口噤不开，四肢抽搐等，《金匮》有专门的篇章来分析)
22. 86 衄家，不可发汗，汗出必额上陥，脉急紧，直视不能眴，不得眠
    > * 禁忌症之一
    > * "额上陥"，历代注家多解，郝认为此处说的是亡阴。亡阴失(脱)水，皮肤没有弹性；
    > * "脉急紧"，亡阴后，脉拘急
    > * "不得眠"，此处"眠"通"瞑"，和第46条一致，是闭目的意思
23. 87 亡血家，不可发汗，发汗则寒栗而振
    > * 禁忌症之一
24. 88 汗家，重(二声)发汗，必恍惚心乱，小便已阴疼，与禹余粮丸
    > * 禁忌症之一
    > * "恍惚心乱"，心神不能自主，注意力不集中，此阴阳两伤，心神失养
    > * "小便已阴疼"，阴液不足，尿道涩滞，故而疼痛
    > * "禹余粮丸"，《伤寒》中只有方名未见方药
25. 89 病人有寒，复发汗，胃中冷，必吐蛕
    > * "蛕"，蛔的古体。蛔虫喜温避寒，故而常寄生于小肠。常态下，即使呕吐也不会吐出来。现在既然吐蛔了，说明其体必有寒(下寒)，所以会往温暖的地方跑(心脏温度最高)，故而会顺着往胃部跑，然而胃部环境呈酸性，也不适合，于是就容易被吐出来。因此，吐蛔的现象即表示下焦有寒。
    > 插一句，根据蛔虫的特性，如果病人大便中见死虫，多半是大限 - 蛔虫都养不活，岂有生机？


## 方
### 麻黄汤
1. 麻黄三两，去节，桂枝二两，去皮，炙甘草一两，杏仁七十个，去皮尖。上四味，以水九升，先煮麻黄，减二升半，去上沫，内诸药，煮取二升，去渣，温服八合，覆取微似汗，不须啜粥，余如桂枝法将息
    > * "先煮麻黄"因为麻黄辛温燥烈，而且以古时的用法，多是新鲜即采，没有现在的制片技术。所以先煮，减少其性温燥烈之性，同样因为麻黄辛温，所以不需要另服热粥
    > * 麻桂甘的比例是3:2:1，大致在这和比例上，根据实际情况加减
    > * 杏仁的量换算下来大约是 10 个约 4g，故方中用量约 28g 杏仁
    > * 汉时一合等于现在 20ml，八合约 160 ml
2. 适应证
    1. 太阳病伤寒表实证 - 第3、35、51、52、37条
    2. 太阳与阳明合病(喘，邪气重) - 第36条
    3. 太阳伤寒与衄解 - 第46、47、55条
        * 太阳病自然病程为 7 天，若果 7 天自解，常有三种方式
        * 自汗烧退
        * 自衄(流鼻血)而解，值得注意的是，只有流鼻血才可能是排除邪气的体现，其他地方(如便血)均不可算作太阳排邪气，后世又称之为红汗，因为汗血同源故
        > 这里又可以根据出血量进行细分，具体见讲义
        * 战汗而解，即发热、头身疼时，突然出现寒战(正邪相争)，寒战之后发热(正气奋力祛邪)，其后汗出烧退
3. 禁忌 - 麻黄九禁，虽称麻黄，其实更应该包含桂枝、大小青龙、葛根汤在内的辛温发汗的所有方子
    1. 咽：咽喉干燥者不可发汗。咽喉乃肺胃之门户，足少阴肾分支所过，咽喉干燥，说明肺、胃、肾可能存在阴虚，此时单纯地发汗的话，其必伤阴 - 第83条
    2. 淋："淋家"，素患淋病之人，即小便短赤，小便淋漓不畅(尿频、尿急、尿不畅)，其体下焦湿热居多，湿热、阴伤(虚热)，故而不可单独使用纯辛温燥烈之方。否则助热，迫血妄行，即尿血 - 第84条
    3. 疮："疮家"，久患疮疡(皮肤哟感染性疾病，长期)之人，其体毒热必盛，气血必伤。故而"身疼痛"乃因气血伤而肌肤失养而致，非寒邪所致 - 第85条
    4. 衄："衄家"，平时常常流鼻血之人，其因常损其血，故而阴亏，发汗后，更损其阴，虚火上扰，即两目直视，不可闭目 - 第86条
    5. 血："亡血家"，平时有出血病的人(如月经过多，或者尿血，皮下出血，甚至鼻衄)，其体气血两虚，用此药后，更伤其阳 - 第87条
    6. 汗："汗家"，平素自汗或者盗汗的人，其体亦是阴阳两虚(更偏阳虚)，再用发汗法，用此药后，更伤阴阳 - 第88条
    7. 寒：其人原本内有寒，中阳(多是脾胃)不足，不宜发汗，发汗更伤阳 - 第89条
    8. 尺脉迟：营血不足，脉迟，不用汗法 - 第50条
    9. 尺脉微：里(阳)虚兼表实证，不宜辛温发汗 - 第49条
    10. 总结：阴阳气血俱不足者，湿热、毒热、阴虚热盛、虚热内盛者，均因慎(禁)用汗方

### 葛根汤 - 伤寒兼证方
1. 葛根四两，麻黄三两，去节，桂枝二两，去皮，生姜三两，切，炙甘草二两，芍药二两，大枣十二枚，擘。上七味，以水一斗，先煮麻黄、葛根，减二升，去上沫，内诸药，煮取三升，去渣，温服一升，覆取微似汗，余如桂枝法将息
2. 此方实际上是以桂枝汤作底方，原因是经脉拘挛，多为津液所伤而致，而麻黄汤辛温发汗力强，不宜用做底方
3. 适应症 
   * 寒邪在表，太阳经气不利 - 第31条
   > * 葛根疏通经脉，升阳发表效果非常好，郝提过一种案例：有一病人，高血压，动脉硬化，尤其表现了颈项部肌肉拘紧不柔和，服用葛根汤后拘紧缓解，血压回归正常。葛根汤对这一类高血压效果很好
   > * 60/61 年黄连素(提纯)治痢疾的差异；"青蒿素抗疟原虫的研究"和"抗青蒿素原虫的研究"
   * 太阳与阳明合病(下利或呕，轻) - 第32、33条

### 葛根加半夏汤 - 伤寒兼证方
1. 葛根四两，麻黄三两，去节，桂枝二两，去皮，生姜二两，切，半夏半升，洗，炙甘草二两，芍药二两，大枣十二枚，擘。上八味，以水一斗，先煮麻黄、葛根，减二升，去上沫，内诸药，煮取三升，去渣，温服一升，覆取微似汗，余如桂枝法将息
2. 此方原理与葛根汤同
3. 适应症  - 第32条

### 大青龙汤 - 伤寒兼证方  
1. 麻黄六两，去节，桂枝二两，去皮，生姜三两，切，炙甘草二两，大枣十枚，擘，杏仁四十枚，去皮尖，石膏，如鸡子大，碎。上七味，以水九升，先煮麻黄，减二升，去上沫，内诸药，煮取三升，去渣，温服一升，取微似汗。汗出多者，温粉粉之。一服汗者，停后服。若复服，汗多亡阳遂虚，恶风烦躁，不得眠也
2. 此方发汗力量最强，寒邪之重，需重用麻黄以发其汗；因里热(遍布周身)盛，故用石膏
3. 方义：重用麻黄，启闭发汗，轻用石膏，兼清里热
4. "温粉"，炒热的米碾成的粉，"温粉粉之"就是用这样的米粉敷在体表，堵住汗孔，起到爽身止汗的作用
5. "恶风"，表阳不足，温煦失司；"烦躁"，肢体躁动不宁，此弱阳与盛阴相争，争而不胜，使得肢体躁动不安，如休克前期的躁动不安、肝昏迷早期的躁动、肺心脑病昏迷的躁动、糖尿病酮中毒昏迷前期的躁动，都属于此种原因，且这种烦躁是不自知的。注意：大青龙汤本证的烦躁是因为热扰心神，是自知的，而这种情况是不自知的，需要鉴别清楚
6. 适应症  - 第38、39条
7. 禁忌：<strong>第一次出汗后，短时间内不可再服用此方</strong>(现代输液技术发达，可以作为补充阴液的后备手段)

### 小青龙汤 - 伤寒兼证方
1. 麻黄去节，桂枝去皮，干姜，炙甘草，芍药，细辛各三两，五味子半升，半夏半升，洗。上八味，以水一斗，先煮麻黄，减二升，去上沫，内诸药，煮取三升，去渣，温服一升。若渴，去半夏，加栝楼根三两；若微利，去麻黄，加荛花，如一鸡子，熬令赤色；若噎者，去麻黄，加炮附子一枚；若小便不利，少腹满者，去麻黄，加茯苓四两；若喘，去麻黄，加杏仁半升，去皮尖。且荛花不治利，麻黄主喘，今此语反之，疑非仲景意。
2. 方义：外散表寒，内散水饮
3. 此方整体上是辛温燥裂有余的，虽然用芍药、五味子敛肺、护营，但是力不足平其他诸药，故不可久服。一般寒饮缓解后改用苓桂剂做养护
4. 干姜，细辛，五味子，这三个药是联合应用的：细辛是散的，五味子是收的，干姜是温化胸中寒饮的。细辛对应肺的宣发功能，五味子对应肺的肃降功能
5. 其他讨论：
   1. "细辛不过钱"，实际应是："细辛若单用末，不可过钱，多则气闷塞不通者死"
## 补充
1. 阳明经表证候歌诀：- 载于《医宗金鉴-伤寒心法要诀》
    * 葛根浮长表阳明 - 阳明经(大肠、胃)是多气多血之经，此经受邪，阳气抗邪于表，脉端直，即"表阳明"，宜用葛根汤
    * 缘缘面赤额头痛 - 阳气被郁，故而满脸通红，同时阳明经行于头部，故而头(脑门)疼
    * 发热恶寒身无汗 - 阳明经表证亦是表证，故而具有发热、恶寒、无汗的共同特征，头项强痛的是太阳，额头痛的是阳明
    * 目痛鼻干卧不宁 - 眼、鼻均是阳明经循经处，而"卧不宁"则是因为胃腑气机失调
2. 咳喘往往有下列的病机
    * 水寒射肺 - 痰多且有白色泡沫(较多)，落地则成水
    * 痰热阻肺
    * 邪热壅肺
    * 风寒闭肺
3. 方剂小结
    * 麻黄汤 - 解表发汗，兼宣肺平喘
    * 桂枝汤 - 解表发汗，兼养阴敛营
    * 葛根汤 - 发表，兼升津舒经
    * 大青龙汤 - 发表，兼清烦(里)热
    * 小青龙汤 - 发表，兼温化水饮
